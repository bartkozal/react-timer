## Development

1. Clone the repository:

   ```sh
   $ git clone git@github.com:brtjkzl/react-timer.git
   ```

2. Install dependencies:

   ```sh
   $ yarn install
   ```

3. Start development:

   ```sh
   $ yarn start
   ```

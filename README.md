# react-timer

Timer app in TypeScript

## Development

1. Clone the repository:

   ```sh
   $ git clone git@github.com:bkzl/react-timer.git
   ```

2. Install dependencies:

   ```sh
   $ yarn install
   ```

3. Start development:

   ```sh
   $ yarn start
   ```
